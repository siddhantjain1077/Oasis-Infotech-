# Retail Sales EDA Project

This folder contains a retail sales exploratory data analysis (EDA) project.

## Files
- Oasis_Retail_Sales_EDA_Template.ipynb: Jupyter notebook for analyzing the retail sales dataset.
- retail_sales_dataset.csv: CSV file containing the sales data used for analysis.

## Purpose
The goal of this project is to explore sales patterns, identify trends, and gain insights from the dataset through data visualization and analysis.

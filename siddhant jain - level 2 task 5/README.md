# Next Word Prediction with LSTM

This folder contains a language modeling project for next-word prediction using an LSTM neural network.

## Files
- lstm_next_word.ipynb: Jupyter notebook for training and testing the model.
- lstm_next_word_corpus.txt: Text corpus used as training data.

## Purpose
The goal of this project is to build a model that predicts the next word in a sequence based on the provided text corpus.

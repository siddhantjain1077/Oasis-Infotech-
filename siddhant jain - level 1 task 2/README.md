# Sentiment Analysis Project

This folder contains a sentiment analysis project based on Twitter text data.

## Files
- Oasis_Sentiment_Analysis_Task4.ipynb: Jupyter notebook for building and evaluating a sentiment analysis model.
- twitter_training.csv: Training dataset used to learn sentiment patterns.
- twitter_validation.csv: Validation dataset used to test the model.

## Purpose
The goal of this project is to classify tweets into sentiment categories and evaluate the performance of the analysis workflow.
